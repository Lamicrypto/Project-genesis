# SQL Data Analysis

This project contains SQL exercises completed through Kaggle's Intro to SQL course.

## Skills Demonstrated
- SELECT statements
- Filtering data
- Sorting results
- GROUP BY
- Aggregate functions
- JOIN operations
- Subqueries

## Tools Used
- SQL
- BigQuery

## About

This project demonstrates practical SQL skills used for querying, cleaning, and analyzing structured datasets. SQL is an essential skill for data analysts, AI engineers, and machine learning practitioners.
