# Feature Engineering Project

This project was completed as part of Kaggle's Feature Engineering course.

## Skills Demonstrated
- Creating new features
- Encoding categorical variables
- Handling dates and text features
- Improving model performance through feature engineering

## Tools Used
- Python
- Pandas
- Scikit-learn

## About

Feature engineering is one of the most important stages of machine learning. This project explores different techniques for transforming raw data into meaningful features that improve predictive performance.
